Chaperone (not publish yet)

is a react library which helps and guide the user interacts with UI step by step to have a better user experience in your react components. Chaperone gives you alot of option with such ability to make your user experience alot better. it allow such gesture to tell the user what to do next even without reading the dialog. it is even flexible in any component whether it is forwardRef or not.

Installation #

npm i chaperone
Imports #

import { Chaperone, ChaperoneStep, Trigger } from "chaperone";
Getting Started #

it is the basic usage of chaperone.

import React from "react";
import { Chaperone, ChaperoneStep, Trigger } from "chaperone";
#render(
    <div className="container">  
        <Chaperone>
            <Trigger label="Need Help?" />
            <form className="mt-3">
                <div className="n-form-field">
                    <label htmlFor="gs-email">Email address</label>
                    <ChaperoneStep 
                        index={0} 
                        message="To Login. Firstly, you must type your e-mail here." 
                        placement="top-right">
                        <input 
                            type="email" 
                            className="n-form-control" 
                            id="gs-email" 
                            aria-describedby="emailHelp" 
                            style={{backgroundColor: "transparent"}} />
                    </ChaperoneStep>
                </div>
                <div className="n-form-field">
                    <label htmlFor="gs-pass">Password</label>
                    <ChaperoneStep 
                        index={1} 
                        message="Secondly, you must put your password here." 
                        placement="top-right">
                        <input 
                            type="password" 
                            className="n-form-control" 
                            id="gs-pass" 
                            style={{backgroundColor: "transparent"}} />
                    </ChaperoneStep> 
                </div> 
                <div className="n-form-field"> 
                    <label className="n-form-check">
                        <ChaperoneStep 
                            index={2} 
                            message={<>Thirdly, if you want to remember the app your inputs just click <b>remember me</b>.</>} 
                            placement="left">
                            <input className="n-form-control" type="checkbox" /> 
                        </ChaperoneStep> 
                        remember me
                    </label> 
                </div>
                <ChaperoneStep 
                    isDone 
                    index={3} 
                    message={<>Lastly, you must press <b>Submit</b>.</>}>
                    <button type="button" className="n-btn">Submit</button>
                </ChaperoneStep> 
            </form>
        </Chaperone>
    </div>

ChaperoneStep Singleton #

it is use to make multiple and different id steps in single ChaperoneStep.

import React from "react";
import { Chaperone, ChaperoneStep, Trigger } from "chaperone";
const chaperoneData = [
    {
        index: 0,
        message: <b>Hello there!!</b>,
        find: "img"
    },
    {
        index: 1,
        message: "I'am Fernando.",
        placement: "left",
        find: "#p-name"
    },
    {
        index: 2,
        message: <>aka <b>Mytabworks</b></>,
        placement: "bottom-right",
        find: (children) => children.querySelector("#p-aka")
    },
    {
        index: 3,
        message: "It can even find outside the children scope.",
        placement: "bottom",
        isDone: true,
        find: () => document.querySelector("#chaperonestep-singleton > div")
    }
]
#render(
    <div className="container mb-4">  
        <Chaperone>
            <Trigger label="click here to know me." allowTransition />
            <ChaperoneStep
                navigation={chaperoneData.length}
                placement="top-right"
                propertyMutation={({ index, currentId }, defaultProps) => {
                    return {
                        ...defaultProps, 
                        ...chaperoneData.find(props => props.index === index)
                    }
                }}>
                <div className="d-flex flex-center-y mt-4">
                    <img className="b-round mr-4" src="/images/icons/icon-128x128.png" alt="beard-man"/>
                    <span id="p-name" className="mr-1">Fernando</span>
                    <span id="p-aka">(Mytabworks)</span>
                </div>
            </ChaperoneStep>
        </Chaperone>
    </div>
)

Multiple ID and Chaining #

you can create multiple guide in single Chaperone and you can chain another ID after done or skip.

import React from "react";
import { Chaperone, ChaperoneStep, Trigger } from "chaperone";
#render(
    <div className="container">  
        <Chaperone onDone={({previousId, trigger}) => {
            if(previousId === "guide-email") {
                trigger("guide-password")
            }
        }}>
            <Trigger id="guide-email" label="Email Help?" allowTransition /><br/>
            <Trigger id="guide-password" label="Password Help?" allowTransition /><br/>
            <form className="mt-3">
                <div className="n-form-field">
                    <ChaperoneStep 
                        navigation={2}
                        id="guide-email" 
                        index={0} 
                        message="It is the Email address label." 
                        placement="top-right"
                        allowSkip={false}>
                        <label htmlFor="mi-email">Email address</label>
                    </ChaperoneStep>
                    <ChaperoneStep 
                        isDone
                        navigation={2}
                        id="guide-email" 
                        index={1} 
                        message="you must type your valid e-mail here." 
                        placement="right-top"
                        onDone={({target, children, dialog}) => { 
                            const test = /^\w+@\w+\.[a-z]{2,4}$/.test(children.value)
                            children.style.borderColor = test ? null : 'red'
                            return test
                        }}>
                        <input 
                            type="email" 
                            className="n-form-control" 
                            id="mi-email" 
                            aria-describedby="emailHelp" 
                            style={{backgroundColor: "transparent"}} />
                    </ChaperoneStep>
                </div>
                <div className="n-form-field">
                    <ChaperoneStep 
                        id="guide-password" 
                        index={0} 
                        message="It is the password label." 
                        placement="top-right"
                        allowSkip={false}>
                        <label htmlFor="mi-pass">Password</label>
                    </ChaperoneStep>
                    <ChaperoneStep 
                        isDone
                        id="guide-password" 
                        index={1} 
                        message="you must put your password here." 
                        placement="top-right">
                        <input 
                            type="password" 
                            className="n-form-control" 
                            id="mi-pass" 
                            style={{backgroundColor: "transparent"}} />
                    </ChaperoneStep> 
                </div>  
                <div className="n-form-field"> 
                    <label className="n-form-check">
                        <input className="n-form-control" type="checkbox" /> 
                        remember me
                    </label> 
                </div>
                <button type="button" className="n-btn">Submit</button>
            </form>
        </Chaperone>
    </div>
)

Password Help?

you must be extra careful when using onDone and onSkip in Chaperone component. especially if the component is having some sort of state. or else it will be called infinitely and will cause error in your app. as much as posible use hook useCallback without array dependency to prevent some sort of error any further.

Chaperone With Gestures #

you can create and append gestures.

import React from "react";

#render(

                        <label htmlFor="inputEmail4">Email</label>
                        <ChaperoneStep
                            allowNext={false}
                            allowBack={false} 
                            propertyMutation={({ currentId, index }, defaultProps) => {
                                const props = [
                                    {
                                        index: 0, 
                                        message: "click the email input box.",

                                            return test
                                        },

Chaperone #

It is the chaperone.
